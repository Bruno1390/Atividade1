Como comentar tags HTML ?

.

Para adicionar um comentário em um código HTML, é preciso utilizar as tags próprias para esse propósito. Portanto, todo o texto que estiver delimitado pelas tags "" que significa o fechamento, é considerado comentário, mesmo que o conteúdo se estenda para outras linhas.

2.Como criar um hiperlink ?

Criar um hiperlink para um local no documento atual. Escolha o texto ou imagem que você deseja exibir como um hiperlink pressione (CTRL+K). Você também pode clicar em link no menu de atalho. Em vincular a, clique e colocae neste documento.

O que é hiperlink ?

Um hiperlink - ou simplesmente um link - é uma referência aos dados que cada pessoa pode seguir diretamente clicando, tocando ou passando o mouse durante a navegação. Os hiperlinks criam ligações/vinculos entre endereços na web ou elos (pontes de ligação) entre documentos. - Como fazer um hiperlink no Word ? Para criar um hiperlink no Word, clique em inserir > link. Na caixa texto para exibição, digite o texto no qual as pessoas clicarão.

6.Criar um documento HTML que usa o elemento artigo ?

-

O Elemento HTML Article (< article >) representa uma composição independente em um documento, página, aplicação, ou site, ou que é destinado a ser distribuido de forma independente ou reutilizável, por exemplo, em sindicação.

7.Criar um documento HTML que usa o elemento aside ?

-

A tag aside define um bloco de conteúdo que faz referência ao conteudo principal que o cerca. Normalmente ela fica na lateral e pode conter assuntos relacionados, conteudo publicitario, informações contextuai.
